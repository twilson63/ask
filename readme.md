# Ask Prompt Module

This module makes it easy to add an ask prompt in your
cake files 

## Usage

```
npm install ask

# in your cakefile
ask = require('ask')

task 'atask', 'description', ->
  ask 'question', /.+/, (answer) ->
    console.log answer
```
